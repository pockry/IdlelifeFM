# IDLELIFE FM
---

一个React的FM组件。最小化依赖，只需引入React库即可。

## 特性

* inline svg
* inline style（TO DO）

## 用法

``` html
<!--idlelife FM开始-->
<div id="idlelife-fm"></div>
<script src="/path/to/idlelifefm.js"  type="text/jsx"></script>
<!--idlelife FM结束->
```

## 警告

**不要将它用在移动Web页面上！它并非为移动站点而设计。**
